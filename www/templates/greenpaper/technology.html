New technology

Government as a Platform will probably require new technologies
  • Tech will keep on changing
  • People's expectations will continue to change
  • The web is now truly an app platform. This is huge.
  • Blockchain and similar mean we can have smart registries that are self validating and managed by multiple organizations (see https://www.ethereum.org/). Blockchain, Matrix and alike don't have to be for massively distributed systems, they can make existing federated systems (eg planning applications) work more efficiently whilst maintaining federation.
  • We need to be careful we don't create new silos of data in helping people move data between devices eg why send an email copy of a receipt via a central email service, when you can use emerging webstandards to let the user decide how to share/save something.
  • New methods of authentication (FIDO Alliance and similar), yield new design patterns for progressive authentication and permissions. Frictionless signing in makes composite services simpler (http://blog.memespring.co.uk/2015/02/06/when-sign-in-becomes-frictionless/).
  • The .dat project will help government migrate data to the new world in a transparent and replicable way (ie don't just pay a data migration company). This is a project government should support.
